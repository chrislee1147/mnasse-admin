# 므낫세 어드민 배포 가이드

## Step 1 — Google Sheets 만들기
1. drive.google.com 에서 새 Google Sheets 생성
2. 제목: "므낫세_어드민_DB"

## Step 2 — Apps Script 세팅
1. 만든 시트에서: 확장 프로그램 > Apps Script 클릭
2. Code.gs 파일에 이 폴더의 `Code.gs` 내용 전체 붙여넣기
3. 저장 (Ctrl+S)
4. 함수 목록에서 `setupAll` 선택 후 실행
   - 권한 허용 팝업 → 계속 → 고급 → 안전하지 않음으로 이동 → 허용
5. 실행 후 로그 확인 (보기 > 로그): "✅ 세팅 완료" 확인

## Step 3 — Apps Script 웹앱으로 배포
1. 배포 > 새 배포 클릭
2. 유형: 웹 앱
3. 설정:
   - 설명: 므낫세 어드민 API
   - 다음으로 실행: 나 (본인 계정)
   - 액세스 권한: 모든 사용자
4. 배포 클릭
5. **웹 앱 URL 복사** (https://script.google.com/macros/s/...../exec)

## Step 4 — GitHub Pages 배포
1. github.com 에서 새 repository 생성
   - 이름: mnasse-admin (또는 원하는 이름)
   - Private 권장 (개인 어드민이므로)
2. 이 폴더(admin)의 파일들 업로드:
   - index.html
   - .gitignore
3. Settings > Pages > Source: main branch / root
4. 몇 분 후 URL 생성됨: https://[계정명].github.io/mnasse-admin/

## Step 5 — 어드민 첫 접속
1. GitHub Pages URL 열기
2. 상단에 노란 배너가 뜸: "Apps Script URL을 입력해주세요"
3. Step 3에서 복사한 URL 붙여넣기 → 저장
4. 완료! (URL은 브라우저에 저장됨)

## 코드 업데이트 방법
- index.html 수정 후 GitHub에 push하면 자동 반영
- Apps Script 수정 시: 배포 > 배포 관리 > 새 버전으로 배포

## 주의사항
- Apps Script URL은 외부에 공유하지 말 것
- 시트 데이터는 구글 드라이브에 영구 저장됨
- 어드민은 GitHub Pages URL을 아는 사람만 접근 가능
